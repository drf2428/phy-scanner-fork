# Physeter Scanner

**Status:** Initial scaffold — work in progress

Open-source fork of [Greenbone Community Edition (OpenVAS)](https://github.com/greenbone) used as the internal Attack Surface Management scan engine of the [Physeter](https://physeter.cloud) platform.

## What is this?

Physeter is a hub and correlator of organizational risk analysis sources. This repository hosts the modifications applied on top of Greenbone CE to integrate with Physeter's SaaS backend, including rebranding, agent-to-server communication, and customizations for the tenant on-prem appliance.

## Status

This is the initial scaffold. Source code from the upstream fork will be imported in upcoming releases. The integration design is being developed in the Physeter parent repository.
